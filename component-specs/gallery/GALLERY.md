# Техническое задание: Компонент Gallery

## Обзор

Реализовать нативный React-компонент `Gallery` для отображения списка thumbnail-изображений с эффектами наложения и интерактивностью. Сторонние UI-библиотеки (кроме `antd`) не используются.

---

## Технические требования

### Отображение изображений

- Компонент отображает список thumbnail-изображений размером **80×60 px**.
- Для рендеринга каждого изображения используется компонент **`Image` из библиотеки Ant Design (`antd`)**.
- Перед использованием компонента `Image` — **обязательно обратиться к MCP context7** для получения актуальной документации по API.
- У компонента `Image` необходимо **отключить дефолтный mask**, передав соответствующий пропс `preview`:

```tsx
<Image width={80} height={60} src={src} preview={{ mask: false }} />
```

---

### Режим "наложения" (более 10 изображений)

Активируется, если количество изображений **больше 10**.

- Каждое изображение **перекрывает ровно половину** следующего (эффект «внахлёст»).
- При наведении курсора на thumbnail:
  - увеличивается `scale` (например, `transform: scale(1.2)`),
  - повышается `z-index` (элемент «выезжает» на передний план).
  - Переход должен быть плавным (`transition`).

---

### Режим "сетки" (10 и менее изображений)

Активируется, если количество изображений **≤ 10**.

- Эффект наложения **отсутствует** (нет отрицательного `margin`).
- Между изображениями добавляется **`gap: 10px`**.

---

### Скролл

- Галерея отображает **максимум 10 изображений** одновременно.
- Если изображений больше 10 — появляется возможность прокрутки.
- **Скролл триггерится только по клику на крайние изображения** в видимой области:
  - клик на **крайнее левое** — прокрутка влево,
  - клик на **крайнее правое** — прокрутка вправо.
- Стандартный скролл мышью/тачпадом **не используется**.

---

## Поведение и состояния

| Условие                 | Поведение                                  |
| ----------------------- | ------------------------------------------ |
| Кол-во изображений ≤ 10 | Сетка с `gap: 10px`, без наложения         |
| Кол-во изображений > 10 | Наложение (50%), скролл по крайним         |
| Hover на thumbnail      | `scale` ↑, `z-index` ↑, плавный transition |
| Клик на крайнее левое   | Прокрутка к предыдущим изображениям        |
| Клик на крайнее правое  | Прокрутка к следующим изображениям         |

---

## Стек и ограничения

- **Фреймворк:** React (TypeScript).
- **Разрешённые библиотеки:** только `antd` (компонент `Image`).
- **Сторонние UI-библиотеки:** не использовать.
- **CSS:** нативный CSS / CSS Modules / inline styles — на усмотрение разработчика.
- **MCP context7:** использовать для проверки актуального API компонента `Image` перед реализацией.

## Критерии приёмки

- [ ] Компонент корректно переключается между режимом сетки и режимом наложения в зависимости от количества изображений.
- [ ] Эффект hover работает плавно (`scale` + `z-index`).
- [ ] Скролл работает только по клику на крайние изображения.
- [ ] Дефолтный mask у `Image` отключён.
- [ ] Размер thumbnail строго 80×60 px.
- [ ] Сторонние библиотеки (кроме `antd`) не используются.
