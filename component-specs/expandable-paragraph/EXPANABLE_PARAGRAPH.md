# Цель проекта

Создать обертку над `Typography.Paragraph` из **Ant Design V5.x** для реализации логики "читать больше/свернуть" с автоматическим определением длины текста и двусторонней кнопкой управления.

## Описание функционала

### Свернутое состояние
- Текст обрезается после **N строк** (по умолчанию 3).
- В конце появляется кнопка **"Показать всё"**.

### Развернутое состояние
- Контейнер растягивается на всю высоту текста.
- Кнопка меняется на **"Свернуть"**.

### Автоматическое поведение
- Короткий текст (**< 4 строк**) — кнопки не отображаются.

## Интерфейс компонента

```typescript
interface ExpandableTextProps {
  /** Исходный текст */
  children: React.ReactNode;
  
  /** Количество строк в свернутом состоянии */
  rows?: number;
  
  /** Начальное развернутое состояние */
  defaultExpanded?: boolean;
  
  /** Управляемое состояние развернутости */
  expanded?: boolean;
  
  /** Обработчик клика по кнопке */
  onExpand?: (expanded: boolean, info: { x: number; y: number }) => void;
  
  /** Текст кнопки (строка или функция) */
  symbol?: string | ((expanded: boolean) => string);
  
  /** Дополнительные CSS классы */
  className?: string;
}
